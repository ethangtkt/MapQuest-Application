About The Class

CSE383 - Web Application Programming

CSE383 was a course I took my Junior year of college. The course consisted of creating full stack web applications (like this one) from the ground up. We dabbled into HTML, PHP, CSS, Bootstrap, and AWS services. Combining all of those things together allowed me to learn what goes into creating websites that we use every day!

MapQuest Application

This project was my final project within the class. The project will show off all the skills I acquired over the 15-week course—everything from bootstrap to working with AWS. More specifically, we are using MapQuest's API to get directions for this website to display. There will also be another feature that will allow us to tap into a database and see every query sent to MapQuest using our application.
